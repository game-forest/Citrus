[EN | RU]

Tangerine main window

  1. Window caption - shows names of opened project and editor.
  2. Menubar - provides editor menu.
  3. Toolbar panel - provides several available tools for working in editor. Configurable from preferences.
  4. Panel «Timeline» - allows you to animate widgets, watch preview of animations and so on.
  5. Panel «Inspector» - provides an interface to view and change properties of the selected widget or group of widgets.
  6. Panel «Viewport» - also known as «Scene View». Provides preview of the current scene and allows you to manipulate widgets in it. Hidden when no documents are opened. Has a status bar at the bottom.
  7. Panel «Filesystem» - file manager for currently opened project. You can use it for navigation between your documents.
  8. Panel «Hierarchy» - provides tree representation of the scene with display filtering feature.
  9. Panel «Backups» - manages documents backups.
  10. Panel «Console» - Tangeine uses it for displaying service and debug information.
  11. Panel «Model3D Attachment» - provides a list of settings for importing 3D-models.
