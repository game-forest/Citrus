Asset cooking

Raw assets

Assets with following extensions are copied as is and packed with deflate:

  • .json
  • .cfg
  • .csv
  • extensions specified using RawAssetExtensions field of citrus project configuration file.
  • .txt except cooking rules and model 3d attachment files.

Assets with following extensions are copied as is and not packed:

  • .ttf
  • .otf
  • .ogv
  • .shader
  • .xml
  • .raw
  • .bin

Note

The sprite data can be accessed in code through ISprite.
